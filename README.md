AI-Powered Student Assistant Chatbot Using Python

A desktop-based AI chatbot application developed using Python and Tkinter GUI for the subject of Artificial Intelligence. This chatbot is designed to interact with users through a simple and user-friendly graphical interface.

Features
Interactive chatbot GUI
Greetings detection
AI-related question answering
Date and time functionality
Simple calculator support
Delete chat feature
Enter key support
Rule-based AI response system
